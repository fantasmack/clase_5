FORMS (FORMAS)

El elemento HTML form representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web

Los componentes más usuales que contiene form son los siguientes:

  • Atributo Action
  • Atributo Method
  • Inputs:
    • Tipo texto
    • Tipo Email
    • Tipo Password
    • Tipo Número
    • Tipo fecha
    • Más Info
  • Textarea
  • Select
  • Button
